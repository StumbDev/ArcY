WebArcy Language

A modern, declarative language for building web applications with simplicity and power. 

// Basic WebArcy example
@page Home {
    @title "Welcome to WebArcy"
    @meta description="A modern web framework"
    
    @component Header {
        @style {
            bg: linear-gradient(45deg, #6649b8, #4a3b8b)
            padding: 2rem
            text-align: center
        }
        
        @h1 { @title }
        @p { "Build beautiful web apps with ease" }
    }
}

Basic Syntax

WebArcy uses a clean, intuitive syntax with directives marked by @ symbols.

Directives

@page
@component
@style
@if
@foreach

Properties

@title: "Page Title"
@meta description="Description"
@prop items: ["a", "b", "c"]

Control Flow

@if condition {
    // content
}

@foreach item in items {
    // content
}

Component System

WebArcy's component system allows you to build reusable UI elements with encapsulated styles and logic.

@component Button {
    @prop type: "primary"
    @prop onClick: () => {}
    
    @style {
        bg: var(--button-${@type}-bg)
        color: white
        padding: 0.5rem 1rem
        border-radius: 4px
        cursor: pointer
    }
    
    @onClick { @prop.onClick() }
    
    slot // Content goes here
}

Usage Example

@component Header {
    @Button type="primary" {
        "Click Me"
        @onClick { alert("Hello!") }
    }
}

Styling System

WebArcy provides a powerful, scoped styling system with CSS-like syntax.

Basic Styling

@style {
    color: #333
    font-size: 16px
    margin: 1rem
    
    &:hover {
        color: #666
    }
}

Variables & Themes

@theme {
    --primary-color: #6649b8
    --text-color: #333
}

@style {
    color: var(--text-color)
    bg: var(--primary-color)
}
