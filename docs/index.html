Installation

Get started with Arcy by following these installation steps:

1. Clone the Repository

git clone https://github.com/yourusername/arcy.git
cd arcy

2. Install Dependencies

yarn install

3. Build the Project

yarn build

4. Start Arcy

yarn start

System Requirements

  • Node.js 16 or higher
  • Yarn package manager
  • Git
